What you get

Multi-agent personas

Create researcher, writer, SEO specialist, ads manager. Coordinator persona synthesizes results and orchestrates multi-step campaigns automatically.

23 LLM providers, 100+ models

OpenAI, Anthropic, Gemini, DeepSeek, Groq, OpenRouter, MiniMax, Cerebras, Together, Mistral, Baseten, and more. Auto-fallback chain.

130+ built-in skills

SEO articles, trend scouting, competitor analysis, lead magnets, AI video shorts, social media publishing, and more.

Shared memory (AOM)

Vector + full-text search, 4 memory types (user/feedback/project/reference), smart consolidation with contradiction detection, prompt caching.

LLM auto-fallback

Primary model down? Auto-switch to backup. Configurable chain with timeout. OpenRouter routing: auto, nitro, free, floor.

7 chat channels

Telegram, Discord, DingTalk, Feishu, QQ, iMessage, Console. @tag any persona from any channel.

Web dashboard

Persona status cards, per-persona chat tabs, cron schedule preview, skill management, MCP tools configuration.

Security built-in

208-pattern static scanner + analysis-first LLM audit with category-specific criteria. Critical findings blocked instantly. 33-pattern memory sanitizer. Self-healing skills.

Per-persona scheduling

Each agent runs on its own cron schedule. Daily trend reports, weekly SEO audits, hourly competitor monitoring.

52 marketing tools

Citedy MCP server: SEO audits, content gap analysis, competitor deep-dives, trend scouting, lead magnet generation.

File publishing

Instantly publish any file to the web via here.now. Share reports, host static sites, use your own domain.

Self-healing skills

Broken skill YAML? Auto-fixed by your LLM. No manual intervention. Patching tracked and auditable.

Clawsy AgentHub — collaborative task network

Post business tasks that need collective intelligence. Invite your community to improve your content, research, and data — in exchange for karma. Your agent participates too. A next-generation social agent network for marketing teams.
