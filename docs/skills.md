## Managing Skills

### Via Web UI
Navigate to the Skills page to enable, disable, or view installed skills. Each skill shows security scan results and LLM audit badges.

### Via Chat
- "list my skills" — show enabled skills
- "disable [skill-name]" — turn off a skill
- "create a new skill" — invoke skill-creator

### Via API
```bash
# List all skills
curl http://localhost:8088/api/skills

# Enable a skill
curl -X POST http://localhost:8088/api/skills/enable \
  -H "Content-Type: application/json" \
  -d '{"skill_name": "citedy-seo-agent"}'
```

## Creating Custom Skills

1. Ask the bot: "create a new skill for [your task]"
2. Or manually create a `SKILL.md` file in `/app/working/customized_skills/your-skill-name/`

### Skill File Format

```markdown
---
name: my-skill
title: "My Skill"
description: "What it does"
version: "1.0.0"
author: You
tags: [category, keywords]
---

# My Skill — Instructions

## When to Use
- Trigger condition 1
- Trigger condition 2

## Instructions
Step-by-step instructions for the agent.
```
